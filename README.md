# DOC - Medical Triage Assistant

## Descarga rápida

Descargá **DOC.exe** desde la sección Releases de este repositorio.

## Cómo usar

1. Descargá DOC.exe de Releases
2. Ejecutalo (doble click)
3. Abrí http://localhost:8080

## Requisitos
- Windows 10/11 64-bit
- Python no requerido (versión portable)

## Stack
- FastAPI + Python 3.12
- Savant Medical Engine
- SQLite (WAL mode)
- JWT + HMAC security
